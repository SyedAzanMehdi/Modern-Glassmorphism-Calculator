# Modern Glassmorphism Calculator
Designed and developed a modern calculator application using HTML, CSS, and JavaScript with a glassmorphism-inspired user interface. The project features arithmetic operations, responsive button layout, blur effects, and clean visual styling.

# Key Features:

* Basic arithmetic operations (+, −, ×, ÷)
* Responsive grid-based button layout
* Glassmorphism UI with blur background effect
* Interactive button handling using JavaScript
* Clean and modern user experience

# Technologies Used:
HTML, CSS, JavaScript

# Concepts Learned:
CSS Grid, Event Delegation, DOM Manipulation, String Logic, UI Design
